# cybersecurity-intrusion-detection
Machine Learning Project - Cybersecurity Intrusion Detection using SVM, Random Forest &amp; Regression
Project Overview

This project uses Machine Learning techniques to detect cyber attacks and classify network traffic as normal or malicious.
The system analyzes cybersecurity data, trains classification models, and visualizes results through an interactive dashboard.

Features
Data preprocessing and cleaning
Label encoding and feature scaling
Machine Learning model training
Intrusion detection classification
ROC curve analysis
Confusion matrix visualization
Feature importance analysis
Cybersecurity dashboard visualization
Technologies Used
Python
pandas
NumPy
Matplotlib
Seaborn
scikit-learn
Machine Learning Algorithms:
Random Forest
SVM
Machine Learning Models
